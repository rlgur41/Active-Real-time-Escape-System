한림대학교 캡스톤 디자인 AREGS(실시간 대피 유도 시스템)

기존 비상구는 고정된 방향을 지시하지만, AREGS의 비상구는 화재, 재난 상황에 따라 비상구의 유도방향을 자동, 수동으로 설정할 수 있다.



팀원 : 박지후, 김병운, 김기혁



수상이력 : 한림대학교 캡스톤 디자인 경진대회 동상
